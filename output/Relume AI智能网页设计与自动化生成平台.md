**【定位：不是网页生成器，而是设计工作流的“加速器”】**

**很多人误以为 Relume 是直接生成 HTML 代码或成品网站的 AI 平台，这是极大的误解。它的真正定位是目前全球最顶尖的 AI 驱动 UI/UX 线框图（Wireframe）与网站地图（Sitemap）生成利器。它不负责最终的视觉皮肤，而是专注于解决建站早期最耗时的“骨架搭建”痛点。**



**【底层逻辑：LLM 语义解析 + 结构化组件映射】**

**Relume 的强悍并非源于视觉大模型，而是通过定制的 LLM API 玩转了结构化数据映射。用户输入一句 Prompt（如“B2B SaaS 落地页”），AI 会将其解析为结构化的 JSON 数据，并精准匹配 Relume 官方维护的 1000+ 个标准企业级组件（导航栏、定价表、FAQ 等）。这种基于规范组件的拼接，保证了 AI 生成的布局绝不“翻车”。**



**【硬核实战：无缝闭环的工作流】**



**1.AI 建模： 输入需求，AI 在几秒内拉出逻辑严密的多页面 Sitemap。**



**2.一键线框图： 确认架构后，AI 自动调用遵循现代 Web 规范的无样式组件，渲染出完整的 Wireframe 并填充好文案。**



**3.生态降维打击（核心）： 这是它最强的地方。生成的线框图可以通过插件，一键复制到 Figma（完美保持矢量格式与自动布局 Auto Layout），或一键复制到 Webflow（完美遵循专业的 Client-First 类名命名规范）。**
